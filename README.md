# Setup
![alt text](http://url/to/https://github.com/JSH4a/Claw/blob/main/claw-frontend/src/assets/logo.png)


At the project root start the nix shell 

    nix-shell shell.nix

Then do

    cd claw-frontend
    npm install

To run the app, do
    
    cargo tauri dev
